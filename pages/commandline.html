Command line utilities¶

ctwrap¶

Wrapper for batch simulations (ctwrap).

usage: ctwrap [-h] [--output OUTPUT] [-v] [--parallel]
              [--strategy {sequence,matrix}]
              module_name yaml_config

Positional Arguments¶

module_name

name of wrapped simulation module

yaml_config

yaml configuration file

Named Arguments¶

--output

name of output file

-v, --verbosity

verbosity level

Default: 0

--parallel

run parallel calculations

Default: False

--strategy

Possible choices: sequence, matrix

batch job strategy
