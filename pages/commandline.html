Command line utilities¶

ctwrap¶

Wrapper for batch simulations (ctwrap).

usage: ctwrap [-h] {list,run} ...

Positional Arguments¶

command

Possible choices: list, run

ctwrap sub-commands

Sub-commands:¶

list¶

List available simulation modules

ctwrap list [-h]

run¶

Run simulation module

ctwrap run [-h] [--output OUTPUT] [-v] [--parallel] [--strategy STRATEGY]
           module_name yaml_config
Positional Arguments¶
module_name

name of wrapped simulation module

yaml_config

yaml configuration file

Named Arguments¶
--output

name of output file

-v, --verbosity

verbosity level

Default: 0

--parallel

run parallel calculations

Default: False

--strategy

batch job strategy
