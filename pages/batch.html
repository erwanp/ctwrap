Custom Simulations¶

In addition, ctwarp supports custom batch simulation. In general, a custom batch job requires a simulation module some_simulation.py and a configuration file batch_configuration.yaml. The corresponding batch job is run as:

$ ctwrap run some_simulation.py batch_configuration.yaml --parallel

Depending on the configuration, results are written to a single file some_simulation.h5 or some_simulation.csv.
